# Парсинг сайта РИА Новости

## Автоматизированный сбор и анализ тональности новостей РИА Новостей за 2023 год

Проект по сбору, структурированию и анализу тональности новостных публикаций с сайта РИА Новости по заданному ключевому запросу (например, «компания ВК»).

🔍 **Задача**:
- Ежемесячно за 2023 год собрать все новости, соответствующие поисковому запросу;
- Извлечь метаданные (дата, заголовок, URL) и полный текст каждой статьи;
- Провести анализ тональности текстов: определить долю позитивных, негативных и нейтральных публикаций.

⚙️ **Реализация**:
- Парсинг поисковой выдачи с интерактивным выбором периода через календарь с использованием `Selenium` (обработка динамического контента, ожидание элементов, scroll-навигация).
- Извлечение текста статей напрямую через requests + `BeautifulSoup` (классы `article__text`, `article__quote-text`).
- Удаление дубликатов по уникальному URL.
- Анализ тональности с помощью библиотеки `dostoevsky` и предобученной модели `FastTextSocialNetworkModel` (многоклассовая классификация: `positive`, `negative`, `neutral`, `speech`, `skip`).

🛠️ **Технологии**:
`Python`, `selenium`, `BeautifulSoup`, `requests`, `pandas`, `dostoevsky`, `lxml`.

📌 **Особенности**:
- Код реализован помесячно для корректной работы с календарным фильтром и пагинацией;
- Встроена обработка ошибок (статусы HTTP, исключения при парсинге);
- Поддержка пауз между запросами для соблюдения этики веб-скрапинга.
