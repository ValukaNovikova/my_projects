# Проекты, выполненные для личного пользования с целью автоматизации процессов

## **`parsing_ria`** - Парсинг и анализ тональности новостей с сайта РИА Новости
Автоматизированный сбор и анализ публикаций с сайта РИА Новости за 2023 год по заданному ключевому запросу. **Реализовано**:
- помесячный парсинг поисковой выдачи с интерактивным выбором периода через `Selenium`;
- извлечение метаданных (дата, заголовок, ссылка) и полного текста статей с помощью `BeautifulSoup`;
- обработка дубликатов и ошибок, соблюдение этики веб-скрапинга;
- анализ тональности текстов с использованием предобученной модели `dostoevsky` (`FastTextSocialNetworkModel`).

*Инструменты**: `Python`, `Selenium`, `BeautifulSoup`, `requests`, `pandas`, `dostoevsky`.

## **`selecting_amount`** - Подбор комбинаций чисел под заданную сумму
Разработано десктопное приложение с графическим интерфейсом для поиска всех комбинаций чисел, сумма которых попадает в заданный диапазон (целевая сумма ± точность). Поддерживает ввод чисел через буфер обмена, настройку максимального размера комбинации, отмену поиска и многопоточную обработку для сохранения отзывчивости интерфейса.

**Инструменты**: Python, `tkinter`, `itertools`, многопоточность (`threading`).
