# Парсинг сайта РИА Новости

## Десктопное приложение для поиска комбинаций чисел с заданной суммой

Это приложение помогает решить распространённую практическую задачу: *"Какие числа из списка в сумме дают указанное значение?"*

Особенно полезно при работе с финансовыми данными: сверка проводок, подбор платежей под счёт, анализ расхождений и т.д.

## 🔍 **Возможности**
- Ввод чисел: через копирование/вставку (Ctrl+V) из Excel, таблиц, текстовых файлов
- Поддержка форматов:
    - каждое число с новой строки
    - числа через `;` (точку с запятой)
    - запятая как десятичный разделитель (автоматически конвертируется в точку)
- Настройка:
    - целевая сумма (например: `34,81`)
    - точность (дельта, например: `0,01` → ищет в диапазоне ±0.01)
    - ограничение на количество чисел в комбинации (0 - без ограничений)
- Многопоточный поиск: интерфейс остаётся отзывчивым даже при переборе тысяч комбинаций
- Остановка поиска в любой момент
- Контекстное меню и горячие клавиши (Ctrl+V, Enter)

## Как запустить
1. Убедитесь, что установлен Python 3.7+
2. Запустите приложение: `python selecting_amount.py`

## Пример использования
**Задача**: Есть список сумм платежей: 12,50; 7,30; 9,45; 15,56; 5,00

Нужно найти, какие из них в сумме дают **22,25** с точностью **0,01**.

**Действия**:
1. Скопируйте числа → вставьте в верхнее поле (Ctrl+V)
2. В поле *"Целевая сумма"* введите: `22,25`
3. В поле *"Точность"* введите: `0,01`
4. Нажмите *"Начать поиск"*
5. В результатах вы увидите:
```
Сумма: 22.250
Комбинация: 15.560 + 7.300
--------------------------------------------------
Сумма: 22.250
Комбинация: 12.500 + 9.450 + 5.000
--------------------------------------------------
```

## Примечания
- При большом количестве чисел (>20) и отсутствии ограничения на размер комбинации поиск может занять время (количество комбинаций растёт экспоненциально).
- Рекомендуется использовать ограничение `Макс. чисел в комбинации` (например, `3` или `4`) для ускорения поиска.
- Все операции выполняются локально - данные не передаются в сеть.
